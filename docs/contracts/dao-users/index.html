    UserDatabase

    Author: Andreas Olofsson (androlo1980@gmail.com)

    • destroy(address)

      Destroy a contract. No return values since it's a destruction.

      Params
      • fundReceiver: (address) the account that receives the funds.

    • dougAddress()

      Get the address of the Doug contract.

      Returns
      • dougAddress: (address) the doug address.

    • hasUser(address)

      Check if a user exists.

      Params
      • addr: (address) the address.

      Returns
      • has: (bool) whether or not the user exists.

    • hasUsers(address,address)

      Check if two users exists.

      Params
      • addr1: (address) the address of the first user.
      • addr2: (address) the address of the second user.

      Returns
      • has1: (bool) whether or not the first user exists
      • has2: (bool) whether or not the second user exists

    • registerUser(address,bytes32,uint256,bytes32)

      Register a new user.

      Params
      • addr: (address) the address.
      • value_dataHash: (bytes32) hash of the file containing (optional) user data.
      • value_nickname: (bytes32) the user name.
      • value_timestamp: (uint) the unix timestamp.

      Returns
      • error: (uint16) error code.

    • removeUser(address)

      Remove a user.

      Params
      • addr: (address) the user address.

      Returns
      • error: (uint16) error code.

    • setDougAddress(address)

      Set the address of the Doug contract.

      Params
      • dougAddr: (address) the address

      Returns
      • added: (bool) true means the address was added successfully (doug implementations should normally not register a contract that returns false.

    • size()

      Get the number of users.

      Returns
      • size: (uint) the number of users.

    • updateDataHash(address,bytes32)

      Register a new data hash for a user.

      Params
      • addr: (address) the address.
      • dataHash: (bytes32) hash of the file containing user data.

      Returns
      • error: (uint16) error code.

    • user(address)

      Get user data.

      Params
      • addr: (address) the address.

      Returns
      • value_nickname: (bytes32) the nickname
      • value_timestamp: (uint) the time when the user was added
      • value_dataHash: (bytes32) the data-hash (optional).

    • userAddressFromIndex(uint256)

      Get a user address by its index in the backing array.

      Params
      • index: (uint) the index.

      Returns
      • key: (address) the key
      • error: (uint16) error code
