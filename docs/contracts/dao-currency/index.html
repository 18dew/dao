AbstractMintedCurrency

Author: Andreas Olofsson (androlo1980@gmail.com)

  • destroy(address)

    Destroy a contract. Calls 'selfdestruct' if caller is Doug.

    Params
    • fundReceiver: (address) the account that receives the funds.

  • dougAddress()

    Get the address of the Doug contract.

    Returns
    • dougAddress: (address) the doug address.

  • mint(address,uint256)

    Mint new coins and add to an account.

    Params
    • amount: (uint) the amount
    • receiver: (address) the receiver account

    Returns
    • error: (uint16) error code

  • minter()

    Get minter account address.

    Returns
    • minter: (address) the minter

  • send(address,uint256)

    Send coins from caller to receiver.

    Params
    • amount: (uint) the amount.
    • receiver: (address) the receiver account

    Returns
    • error: (uint16) error code

  • setCurrencyDatabase(address)

    Set the address of the currency database. Can only be done by the minter.

    Params
    • dbAddr: (address) the database address.

    Returns
    • error: (uint16) error code

  • setDougAddress(address)

    Set the address of the Doug contract. Will fail if address is already set, or if input address is 0.

    Params
    • dougAddr: (address) the address

    Returns
    • added: (bool) true means the address was added successfully. Doug implementations should normally not register a contract that returns false.

  • setMinter(address)

    Set the minter address.

    Params
    • minter: (address) the minter

    Returns
    • error: (uint16) error code

CurrencyDatabase

Author: Andreas Olofsson (androlo1980@gmail.com)

  • accountBalance(address)

    Get the balance of an account.

    Params
    • addr: (address) the account address

    Returns
    • balance: (uint) the balance

  • add(address,int256)

    Add currency to an account.

    Params
    • amount: (int) the amount. Use a negative value to subtract.
    • receiver: (address) the receiver account

    Returns
    • error: (uint16) error code.

  • destroy(address)

    Destroy a contract. No return values since it's a destruction. Calls 'selfdestruct' on the contract if successful.

    Params
    • fundReceiver: (address) the account that receives the funds.

  • dougAddress()

    Get the address of the Doug contract.

    Returns
    • dougAddress: (address) the doug address.

  • send(address,address,uint256)

    Send currency between accounts.

    Params
    • amount: (uint) the amount.
    • receiver: (address) the receiver account
    • sender: (address) the sender account

    Returns
    • error: (uint16) error code.

  • setDougAddress(address)

    Set the address of the Doug contract. Will fail if address is already set, or if input address is 0.

    Params
    • dougAddr: (address) the address

    Returns
    • added: (bool) true means the address was added successfully. Doug implementations should normally not register a contract that returns false.
